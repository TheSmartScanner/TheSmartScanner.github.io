Yet another vulnerability scanner

SmartScanner is an automated web vulnerability scanner you can use to test the security of websites and web applications. Unlike most security scanners, SmartScanner is taking advantage of Artificial Intelligence (AI) and machine learning methods to reach and find vulnerabilities where other scanners cannot detect most of the time. This means that SmartScanner has a minimum number of false positives and false negatives for security issues it can test and it gets better over each upgrade!

How Smart is it?

Certainly not as much as it could replace an experienced penetration tester but it is always learning and can be a perfect assist. Many security issues are as easy as detecting a static simple pattern. For example detection of a missing Http-only cookie attribute, is easy but detection of a session relevant cookie might not be that easy for an automated tool. Business logics are another example that is hard for automated tools to understand.
