Installation

Installing SmartScanner is straight forward.

  • Download SmartScanner
  • Run the installer
  • Follow the installation steps

Note: SmartScanner will be installed for current user only. If you need to install SmartScanner Professional for multiple users, you have to purchase multiple licenses.

Make sure you have the below minimum requirements before installing SmartScanner.

  • 500MB of free disk space
  • 2GM of free RAM
  • Windows 7 or later versions
