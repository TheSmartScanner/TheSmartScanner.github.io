How does it work?

SmartScanner is a dynamic application security testing (DAST), also known as “black-box” testing tool. It means you need an up and running website or web application for testing. SmartScanner is fully automated, all you have to do is to enter the web site’s URL and start scanning! SmartScanner will automatically crawl website pages, execute JavaScripts, and identify attack vectors such as parameters and cookies. Then, SmartScanner tests for a wide range of security issues. You can select tests before a scan and configure different parameters like authentication and proxy as well.

Unlike most of the security scanners, SmartScanner takes advantage of artificial intelligence. it learns about the website behavior during the scan and can identify issues that other scanners miss most of the time.
